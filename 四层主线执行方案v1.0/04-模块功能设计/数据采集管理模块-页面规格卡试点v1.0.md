## 页面卡A：批量导入页面

### 1. 基本信息
- 模块名称：数据采集管理
- 页面名称：批量导入
- 页面编号：M2-P01
- 状态：Ready for Dev

### 2. 页面目标
- 完成批次导入并可追踪导入结果。
- 成功判定：导入结果可查看、失败原因可定位、批次可追溯。

### 3. 入口与前置条件
- 入口：数据采集管理 > 批量导入
- 前置条件：用户具备导入权限
- 依赖能力：权限校验、文件存储、批次记录

### 4. 状态定义

| 状态 | 触发条件 | 页面表现 | 用户动作 |
|------|----------|----------|----------|
| 空状态 | 首次进入或无记录 | 展示上传引导 | 上传文件 |
| 加载中 | 上传/校验进行中 | 显示进度 | 等待/取消 |
| 成功态 | 导入完成 | 展示成功失败统计 | 查看批次详情 |
| 异常态 | 校验失败或服务错误 | 显示错误原因 | 修正后重试 |
| 无权限态 | 无导入权限 | 拒绝访问提示 | 返回 |

### 5. 关键交互流程
1. 选择来源并上传文件。  
2. 系统执行准入校验并返回结果。  
3. 用户查看详情并处理失败项。

### 6. 字段与规则

| 字段/区域 | 类型 | 必填 | 规则 | 错误提示 |
|-----------|------|------|------|----------|
| 数据来源 | 下拉 | 是 | 仅允许预置来源 | 请选择数据来源 |
| 上传文件 | 文件 | 是 | 文件格式和大小必须合规 | 文件不符合要求 |
| 批次备注 | 文本 | 否 | 长度限制 | 备注超长 |

### 7. 异常场景

| 场景 | 触发条件 | 处理策略 | 回流动作 |
|------|----------|----------|----------|
| 格式错误 | 文件结构不合法 | 拒绝入库并提示原因 | 修正文件后重传 |
| 信息缺失 | 关键字段为空 | 阻止提交 | 补齐后重试 |
| 服务不可用 | 校验服务异常 | 给出重试建议 | 稍后重试 |

### 8. 验收条件（Ready）
- [x] 目标与入口明确
- [x] 状态完整
- [x] 字段规则可测
- [x] 异常可复现
- [x] 依赖与影响明确

### 9. 影响与依赖
- 上游依赖：权限与账号管理
- 下游影响：待标注照片池入池质量
- 关联文档：L2-U1、L3-数据采集管理模块

## 页面卡B：导入批次详情页面

### 1. 基本信息
- 模块名称：数据采集管理
- 页面名称：导入批次详情
- 页面编号：M2-P02
- 状态：Ready for Dev

### 2. 页面目标
- 支持批次结果追踪与失败项定位。
- 成功判定：可按批次、状态查看导入记录并定位失败原因。

### 3. 入口与前置条件
- 入口：批量导入结果页 > 查看详情
- 前置条件：存在批次记录
- 依赖能力：批次查询服务、失败原因记录

### 4. 状态定义

| 状态 | 触发条件 | 页面表现 | 用户动作 |
|------|----------|----------|----------|
| 空状态 | 批次无记录 | 空数据提示 | 返回导入页 |
| 加载中 | 查询中 | 加载提示 | 等待 |
| 成功态 | 查询成功 | 展示统计和失败明细 | 筛选/导出 |
| 异常态 | 查询失败 | 错误提示 | 重试 |
| 无权限态 | 无查看权限 | 拒绝访问提示 | 返回 |

### 5. 关键交互流程
1. 进入详情查看统计。  
2. 按成功/失败筛选记录。  
3. 定位失败项并发起重新导入。

### 6. 字段与规则

| 字段/区域 | 类型 | 必填 | 规则 | 错误提示 |
|-----------|------|------|------|----------|
| 批次号 | 文本 | 是 | 系统唯一标识 | 批次不存在 |
| 成功/失败统计 | 只读 | 是 | 成功+失败=总数 | 统计异常 |
| 失败原因 | 标签 | 否 | 来自校验规则 | - |

### 7. 异常场景

| 场景 | 触发条件 | 处理策略 | 回流动作 |
|------|----------|----------|----------|
| 批次失效 | 批次号不存在 | 提示并返回列表 | 重新查询 |
| 查询异常 | 服务不可用 | 提供重试能力 | 稍后重试 |

- [x] 状态完整
- [x] 字段规则可测
- [x] 异常可复现
- [x] 依赖与影响明确

### 9. 影响与依赖
- 上游依赖：批量导入页面
- 下游影响：数据准入问题处理效率
- 关联文档：L2-G1关口、L3-数据采集管理模块
